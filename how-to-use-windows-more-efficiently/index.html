如何高效地使用Windows

发表于 2016-05-21 阅读次数： Valine：

一不小心又断更了大半个月，之前写了一篇《如何优雅地使用Mac》，今天再来写一篇《如何高效地使用Windows》。

日常使用篇

Listary

Listary官网：http://www.listary.com

Listary是一款极为优秀的文件浏览与搜索增强工具。它利用空闲时间在后台建立文件索引，当你需要使用任何文件时，可以瞬间完成全盘查找。不仅可以在资源管理器中查找，还能对所有的文件/目录选择窗口进行快速查找定位。通过Listary，可以极大的提高用户的操作效率。

关于Listary更详细直观的介绍，可以观看由“异次元软件”提供的视频：

Listary是一款共享软件，免费用户部分功能受到限制，接下来你懂的，请支持正版。

Ditto

Ditto官网：http://ditto-cp.sourceforge.net

Ditto是Windows下一款免费开源的剪贴板增强软件，它可以长时间保存剪贴板历史记录，即便你关机或者重启，剪贴板也仍然能够保留。支持历史检索，配合Listary使用会有奇效。

关于Ditto更详细的介绍可以参考“异次元软件”的文章，这里不再赘述。Ditto是一款免费开源的软件，直接在官网下载安装包即可。

HashTab

HashTab官网：http://implbits.com/products/hashtab

HashTab是一款超轻量文件校验工具（支持MD5等多种哈希算法），其程序本身没有入口，通过与Windows资源管理器集成，直接在文件上右键属性窗口中加入HashTab标签页，点击即可计算/比较文件哈希值。

HashTab是一款免费软件，直接在官网下载安装即可。

软媒时间

软媒时间官网：http://sj.ruanmei.com

软媒时间是一款增强Windows时间的工具，可以在任务栏时钟区域直接显示农历和天气，并且提供定时提醒、备忘录等功能。

软媒时间是一款免费软件，直接在官网下载安装即可。

码农进阶篇

Notepad++

Notepad++官网：https://notepad-plus-plus.org

Notepad++是一款超轻量的文本编辑器，运行速度与Windows自带记事本无异，功能强大又不复杂，非常适合作为日常使用的文本编辑器，临时浏览一下代码也是不错。不过专门用于开发就略显简陋了，开发用途的轻量级代码编辑器可以考虑Sublime Text或者Visual Studio Code (Atom)。

Notepad++是一款免费软件，直接官网下载安装即可。

Cmder

Cmder官网：http://cmder.net

Cmder是一款Windows下的终端模拟器，具有支持富文本，支持多标签，集成Git支持等功能，比起Windows十几年不变的cmd命令行，Cmder不知高到哪里去了。

Cmder是一款开源免费绿色软件，直接在官网下载即可。

解压后，可以通过在cmd或者cmder中输入cmder.exe /register all注册到资源管理器右键菜单，这样就可以在任意目录右键打开cmder并自动定位到当前目录了。

SourceTree

SourceTree官网：https://www.sourcetreeapp.com

SourceTree是一款有澳大利亚Atlassian公司开发的图形化Git工具，原生于Mac平台，后作为Mac平台最佳的图形Git工具而移植到Windows。

不过也许是暴力移植的关系，SourceTree在Windows平台下的表现并不是很好，容易出现卡顿甚至卡死无法恢复的情况（Mac平台下不存在类似问题）。但相对而言Windows平台下本身也没有好用的图形化Git，因此还是推荐功能强大的SourceTree作为首选Git工具。

SourceTree需要注册Atlassian帐号即可获得免费使用授权，直接在官网下载安装即可。SourceTree及Atlassian官网偶尔被墙，如遇无法访问请科学上网。

Beyond Compare

BeyondCompare官网：http://www.scootersoftware.com

Beyond Compare是一款老牌的文件比较、合并工具，之前在Mac平台也推荐过。通过Beyond Compare进行代码比较、解决Git冲突非常方便，因此作为程序员必备工具，墙裂推荐。

BeyondCompare是收费软件，提供免费试用。接下来你懂的，请支持正版。

Visual Studio篇

JetBrains ReSharper

ReSharper官网：https://www.jetbrains.com/resharper

ReSharper是一款来自捷克JetBrains的Visual Studio插件，JetBrains是一家著名的制造IDE和软件开发工具的公司，代表作品有IDEA、WebStorm。Android Studio即是基于IDEA定制的。

ReSharper提供了非常强大的辅助功能，支持C#和C++语言，提供模糊补全，代码模版，智能感知（比VS的更智能），代码重构，代码规范提醒，代码优化提醒等功能。在C#语言中还具备非常强大的Linq优化。

ReSharper是一款收费插件，可以免费试用30天。值得一提的是，JetBrains公司对全球的教育用户免费提供所有的产品，因此在校大学生只需要向学校申请一个edu域名的教育邮箱（Ps，某宝上也有人卖教育邮箱），通过教育邮箱绑定JetBrains帐号即可获得所有产品一年期的免费使用授权，到期后只要你的edu邮箱仍然有效，就可以续期，直到毕业（edu邮箱被回收为止）。
